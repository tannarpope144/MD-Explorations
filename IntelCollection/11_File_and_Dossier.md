# 11. File and Dossier

**Tier 3 — Information-posture approaches**
**Source:** FM 2-22.3, paras 8-55–8-57

- **Doctrine:** "We Know All" with a prop — a thick, tabbed file (padded) implying total knowledge. Dependent on target naïveté.
- **LLM analog:** Prop-supported bluff; the "evidence theater" pattern.
- **Research note:** Variant of #10; useful for studying how concrete (even fake) artifacts amplify a persuasion claim's credibility.

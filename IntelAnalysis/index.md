# Intelligence Analysis — Technique Reference

A distilled, LLM-usable reference derived from **ATP 2-33.4, *Intelligence Analysis*** (U.S. Army, 18 August 2014). Each analytic technique is a self-contained file with its definition, when-to-use guidance, method steps, worked examples, tips, and cautions. Companion reference files cover analytical pitfalls, technique selection, and terminology.

**Start here:**
- **Don't know which technique to use?** → [Technique Selector](reference/technique-selector.md) — matches techniques to purpose and operation type (offense/defense, stability, DSCA).
- **Checking your own reasoning?** → [Analytical Pitfalls, Fallacies & Biases](reference/pitfalls.md) — the critical-thinking framework plus every logic fallacy and bias.
- **Need a term or acronym?** → [Glossary](reference/glossary.md).

## Techniques

### Basic structured techniques
The simplest building blocks; the starting point for most analyses.

| Technique | Use it to… |
|---|---|
| [Sorting](techniques/sorting.md) | Group large or multi-category data to surface trends and anomalies. |
| [Matrices](techniques/matrices.md) | Isolate critical data and compare elements in a grid (e.g., PMESII × ASCOPE). |
| [Threat Intentions Matrix](techniques/threat-intentions-matrix.md) | Analyze options from the threat's perspective and derive indicators. |
| [Event Mapping](techniques/event-mapping.md) | Brainstorm and visualize scenario hypotheses radially around a central issue. |
| [Event Trees](techniques/event-trees.md) | Map alternative event sequences and decision points toward an outcome. |
| [Subjective Probability](techniques/subjective-probability.md) | Quantify degree of belief across a complete set of hypotheses. |
| [Weighted Ranking](techniques/weighted-ranking.md) | Rank options transparently by pair-ranking weighted criteria. |

### Diagnostic techniques
Make analytic arguments, assumptions, and intelligence gaps more transparent. Used with most other techniques.

| Technique | Use it to… |
|---|---|
| [Deception Detection](techniques/deception-detection.md) | Decide when to look for deception and avoid being deceived. |
| [Key Assumptions Check](techniques/key-assumptions-check.md) | Make stated and unstated assumptions explicit and testable. |
| [Quality of Information Check](techniques/quality-of-information-check.md) | Evaluate information completeness/soundness independent of the source. |
| [Indicators](techniques/indicators.md) | Develop, weigh, analyze, and use indicators (incl. Appendix B example tables). |
| [Conducting Studies](techniques/conducting-studies.md) | Produce detailed area/regional/country or specified studies. |

### Core Army analytic techniques
Routinely used at strategic, operational, and tactical levels.

*Developing situational understanding and conclusions:*

| Technique | Use it to… |
|---|---|
| [Brainstorming](techniques/brainstorming.md) | Generate a range of hypotheses and break down mindsets (2 phases, 11 steps). |
| [Comparison](techniques/comparison.md) | Weigh options against historical precedents; CARVER target comparison. |
| [Mathematical Analysis](techniques/mathematical-analysis.md) | Assess capabilities/limitations via tangible and intangible ratios. |
| [Situational Logic](techniques/situational-logic.md) | Trace cause-effect / means-end for a situation treated as unique. |

*Analyzing complex networks and associations:*

| Technique | Use it to… |
|---|---|
| [Link Analysis](techniques/link-analysis.md) | Quickly map connections (association/activities matrices, link diagram). |
| [Network Analysis](techniques/network-analysis.md) | Examine network structure, critical nodes, and organizational forms. |
| [Social Network Analysis](techniques/social-network-analysis.md) | Measure centrality, density, and distance; target HVIs via social ties. |

*Conducting pattern analysis:*

| Technique | Use it to… |
|---|---|
| [Pattern Analysis](techniques/pattern-analysis.md) | Predict future activity from past patterns — chronologies/timelines, time event chart, plot sheet, incident overlay, pattern of life (all 5 tools). |

## Notes on this reference

- **Source:** all content distilled from ATP 2-33.4 (18 August 2014). Page furniture and paragraph numbers from the source PDF have been removed for readability; figures/diagrams in the original are not reproduced (they were image-only).
- **Scope:** the 36 analytic techniques plus the three reference files above. Chapter 6/7 mission-context narrative (offense/defense/stability/DSCA support, counterinsurgency, CIED, site exploitation) is not reproduced as standalone files; the operational applicability of each technique is captured in the [Technique Selector](reference/technique-selector.md).
- **Center of gravity analysis, functional analysis, and modeling** are referenced by the source but detailed in FM 2-01.3, not ATP 2-33.4 — they are listed in the selector but have no technique file here.

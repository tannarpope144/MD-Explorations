# Comparison

**Category:** Core Army analytic technique
**Source:** ATP 2-33.4, *Intelligence Analysis*

## Definition

Comparison is used to understand current events by comparing them with historical precedents in the same country or with similar events in other countries. It differs from [applying theory](applying-theory.md) in that conclusions are drawn from a **small** number of cases, whereas applying theory is generated from examining a **large** number of cases.

## When to use

Useful when faced with ambiguous situations — it looks at how the country handled similar situations in the past or how similar countries managed comparable situations. Comparison analysis:

- Helps determine the importance of several options relative to each other.
- Is particularly useful when the analyst lacks objective data on which to base conclusions.
- Simplifies choosing the most important problem to solve, or selecting the solution providing the greatest advantage.
- Assists in setting priorities when there are conflicting demands on resources (paired comparison analysis).
- Is an effective method for weighing the relative likelihood of different COAs based on facts and assumptions, particularly when COAs are ambiguous or of similar likelihood.

## CARVER method

The **CARVER** targeting method (criticality, accessibility, recuperability, vulnerability, effect, recognizability) is used to compare potential enemy targets. Lower scores indicate less desirable outcomes; higher scores indicate more desirable outcomes.

- **In the offensive:** helps identify targets vulnerable to attack.
- **In the defensive:** identifies high-risk targets requiring additional security assets for protection.

CARVER compares the value of a potential target against the risk of neutralizing it. (See ATP 3-05.1 for more on unconventional warfare.)

### Steps

1. List the variables to be compared. Brainstorming can help generate an initial list, then refine it into a final list.
2. Mark the variables as row headings on a matrix.
3. Place the events being compared as column headings.
4. Move across each row and determine the importance of each variable to that event's occurrence by assigning it a weighted rank. Consider each variable in relation to the country, culture, and specific circumstances of the events.
5. After weighting each variable, consolidate each event's weight by summing the columns. If the numbers are similar, the analyst may assess that the event is likely to occur in a manner similar to the event used as a comparison.

The CARVER technique provides a framework for comparing each COA against all others and illustrates the differences in importance between factors.
